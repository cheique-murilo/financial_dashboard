This was one of the dashboards developed during my studies at Hashtag Treinamentos. It is based on a fictional Excel database that feeds the data presentation in the report shared below. The database contains information about payments and receipts from different banks as well as various payment methods (Pix, credit card, boleto, or MercadoPago). It is a simple report, but one where we used different types of charts and Power BI tools (such as the filter with the "Image Grid" chart displaying bank logos, the custom profit margin using a donut chart + card, the automatic scrolling bar with the "Scroller" chart (city margin), and the "enlighten data story" chart - text with dynamic values)..
